# Kullanıcı Odak Takibi (Focus Tracker) - Kullanım Notları

## Windows Kullanıcıları İçin:
- [Sürümler sayfasında](https://github.com/yasinemira/focus-tracker/releases) yer alan en güncel paket sürümü içinde yer alan windows-exe.zip dosyasını çıkardıktan (unzip) sonra, karşınıza çıkan _focus-tracker.exe_ dosyasına çift tıklayarak uygulamayı başlatabilirsiniz.

- Uygulamadan çıkmak için ESC veya Q tuşuna basabilirsiniz.

## MacOS Kullanıcıları İçin:

- [Sürümler sayfasında](https://github.com/yasinemira/focus-tracker/releases) yer alan en güncel paket sürümü içinde yer alan mac-app.zip dosyasını çıkardıktan (unzip) sonra uygulama dosyasına **sağ tıklayın** ve "Paket İçeriğini Göster" (Show Package Contents) seçeneğine basın.
    - Açılan pencerede sırasıyla _Contents_ ve sonrasında _MacOS_ dizinine gidin.
    - Bu klasörün içerisinde bulunan siyah ikonlu uygulama dosyasına çift tıklayarak çalıştırın.

- Uygulamadan çıkmak için **ESC** veya **Q** tuşuna basabilirsiniz.
